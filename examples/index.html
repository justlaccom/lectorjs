LectorJS - Lecteur Vidéo Personnalisable

Lecteur Vidéo

Thèmes prédéfinis :

Code d'Exemple

// Initialisation du lecteur
const player = new Lector('#video-player', {
    width: '100%',
    height: '450px',
    colors: {
        primary: '#2196F3',
        secondary: '#FF4081',
        text: '#ffffff',
        background: '#1a1a1a'
    },
    controls: {
        playPause: true,
        progress: true,
        volume: true,
        fullscreen: true,
        time: true,
        speed: true
    },
    autoplay: false,
    loop: false,
    muted: false,
    preload: 'metadata'
});

// Définir la source vidéo
player.setSource('https://commondatastorage.googleapis.com/gtv-videos-bucket/sample/BigBuckBunny.mp4');
